DEVZI – Educational Landing Page

DEVZI is a simple and visually appealing front-end website designed for an online education platform. It focuses on user engagement with a strong call-to-action and a modern layout.

Features--
Responsive navigation bar (Home, About, Services, Contact)
Eye-catching hero section with headline and image
Lead capture form (mobile number input)
Clean and minimal UI design
Fully built with HTML and CSS

Technologies Used--
HTML5
CSS3

Purpose
This project is created to practice front-end development and design a professional landing page for educational and other platform services.
